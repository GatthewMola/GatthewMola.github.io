Math Painter Game

The Math Painter Game allows the user to draw shapes (squares and rectangles) of various colors onto a canvas. The user is prompted to select starting coordinates for their shapes, along with parameters for the shapes dimensions. The user is allowed to draw as mainly shapes they desire within the specified confines of the canvas.
